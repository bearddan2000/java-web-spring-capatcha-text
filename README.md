# java-web-spring-capatcha-text

## Description
A spring implementation of
capatcha as a simple text.

## Tech stack
- java
- maven

## Docker stack
- maven:3-openjdk-17

## To run
`sudo ./install.sh -u`
- Available at http://localhost

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
